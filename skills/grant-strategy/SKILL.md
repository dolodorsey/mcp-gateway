# Grant Strategy Engine

Grant funding identification, positioning, and proposal system for the Kollective's fundable properties — non-profit initiatives, cultural installations, community programs, and mission-aligned ventures.

## Core Principle

Grant funding is strategic capital, not free money. Every grant comes with obligations — reporting, compliance, timeline adherence, and outcome measurement. The goal is to find high-fit opportunities where the Kollective's existing work naturally aligns with funder priorities, not to reshape programs to chase money.

## Grant Readiness Assessment

### Before applying, verify:

**Entity Readiness:**
- [ ] 501(c)(3) status confirmed (Playmakers) or fiscal sponsor identified
- [ ] EIN (Employer Identification Number) active
- [ ] SAM.gov registration current (for federal grants)
- [ ] Financial statements available (3 years preferred)
- [ ] Board of directors established and documented
- [ ] Organizational budget prepared
- [ ] Previous grant history documented (if any)

**Program Readiness:**
- [ ] Program description written (clear, measurable)
- [ ] Target population defined
- [ ] Geographic area specified
- [ ] Timeline with milestones
- [ ] Budget with line items
- [ ] Evaluation plan (how you'll measure impact)
- [ ] Letters of support (partners, community, beneficiaries)
- [ ] Staff/leadership bios

**Compliance Readiness:**
- [ ] Understand reporting requirements
- [ ] Capable of tracking outcomes
- [ ] Financial management systems in place
- [ ] Audit capacity (if required)
- [ ] Insurance coverage adequate

## Proposal Narrative Structure

### Standard Grant Narrative (adapt per funder requirements)

```
1. EXECUTIVE SUMMARY (1 paragraph)
   Who you are, what you're requesting, what it accomplishes, who benefits.

2. ORGANIZATIONAL BACKGROUND (1-2 paragraphs)
   History, mission, track record, capacity to deliver.

3. STATEMENT OF NEED (1-2 paragraphs)
   The problem or opportunity. Data-supported. Community-specific.
   Why now. Why this matters.

4. PROJECT DESCRIPTION (2-4 paragraphs)
   What you'll do. How you'll do it. Timeline. Activities. Participants.
   Be specific — funders want to see a plan, not a vision statement.

5. GOALS AND OBJECTIVES (measurable)
   Goal: Broad impact statement
   Objective 1: Specific, measurable, time-bound
   Objective 2: Specific, measurable, time-bound
   Objective 3: Specific, measurable, time-bound

6. EVALUATION PLAN (1-2 paragraphs)
   How you'll measure success. What data you'll collect.
   Pre/post metrics. Surveys. Attendance. Outcomes.

7. SUSTAINABILITY (1 paragraph)
   How the program continues after grant funding ends.
   Revenue diversification. Ongoing partnerships. Earned income.

8. BUDGET NARRATIVE (accompanies budget spreadsheet)
   Justify each line item. Show how funds will be used.
   Include match/leverage if applicable.

9. ATTACHMENTS
   Board list, financial statements, letters of support,
   IRS determination letter, organizational chart, resumes.
```

## Budget Framework

### Standard Grant Budget Categories
| Category | Examples | Notes |
|---|---|---|
| Personnel | Staff salaries, contractor fees | Show time allocation % |
| Fringe benefits | Benefits, taxes on personnel | Typically 20-30% of salary |
| Travel | Staff travel to program sites | Per diem rates, mileage |
| Equipment | Durable goods over $5K | Justify necessity |
| Supplies | Materials, consumables | Itemize major categories |
| Contractual | Subcontractors, consultants | Scope of work for each |
| Construction/Renovation | Venue build-out (if applicable) | Requires separate approval often |
| Other | Insurance, permits, licensing | Itemize |
| Indirect costs | Overhead, admin | Many funders cap at 10-15% |

## Output Modes

### MODE 1: Funding Strategy
Comprehensive grant strategy for a specific property:
1. Property assessment (what's fundable)
2. Best-fit grant categories
3. Target funders (10-20 specific opportunities)
4. Positioning angle per funder
5. Readiness gaps
6. 12-month application calendar
7. Budget framework
8. Success probability assessment

### MODE 2: Grant Proposal Draft
Full proposal narrative:
1. Executive summary
2. Organizational background
3. Statement of need
4. Project description
5. Goals and objectives (measurable)
6. Evaluation plan
7. Sustainability plan
8. Budget narrative
9. Required attachments checklist

### MODE 3: Eligibility Assessment
Quick-check eligibility for a specific grant:
1. Funder requirements vs. applicant profile
2. Eligible entity type (non-profit, government, etc.)
3. Geographic eligibility
4. Program area alignment
5. Budget range fit
6. Timeline alignment
7. Go/no-go recommendation with reasoning

### MODE 4: Grant Calendar
Annual application planning:
1. All target grants with deadlines
2. Application preparation timeline (start date per grant)
3. Document preparation schedule
4. Review/submission deadlines
5. Reporting dates (if awarded)
6. Team assignments

### MODE 5: Impact Report
Post-award reporting:
1. Goals vs. actual outcomes
2. Participation data
3. Financial expenditure report
4. Qualitative impact stories
5. Lessons learned
6. Sustainability progress
7. Photos/documentation
