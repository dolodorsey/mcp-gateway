# Creative Production Engine

Visual asset production for every Kollective brand. Every graphic, flyer, menu, logo, and visual deliverable must look like the brand produced it in-house — not like a generic template.

## Core Principle

The Kollective's visual identity IS the brand experience before anyone attends an event, eats the food, visits the museum, or uses the service. Creative assets are the first impression and the lasting memory. Every pixel represents the brand.

## Design Brief Template

```
BRAND: [Exact brand name — never approximate]
ASSET TYPE: [From asset catalog above]
DIMENSIONS: [Width × Height + DPI]
HEADLINE: [Primary text — max 8 words]
SUBTEXT: [Secondary text — date, location, tagline]
BODY COPY: [If applicable — keep minimal]
MUST INCLUDE: [Logo, QR code, handles, partner logos, ticket link]
VISUAL STYLE: [Reference brand visual system above]
MOOD: [1-3 descriptive words]
COLOR OVERRIDE: [Only if different from brand standard — must justify]
IMAGE DIRECTION: [Photo style, illustration style, abstract, etc.]
REFERENCE: [Link to inspiration or past asset]
FILE FORMAT: [PNG / SVG / PDF / JPEG]
DELIVERY: [Google Drive folder / Supabase storage / direct / email]
DEADLINE: [Date + time]
APPROVAL: [Who approves before publish]
```

## AI Image Generation Prompt Template

When generating images via GRAPHICS ENGINE — Nano Banana or any AI image tool:

```
[BRAND]: {brand_name}
[STYLE]: {brand photography/illustration style from visual system}
[SCENE]: {detailed scene description}
[MOOD]: {emotional quality — from brand mood column}
[COLORS]: {dominant: brand primary, accent: brand secondary}
[COMPOSITION]: {layout direction — centered, rule of thirds, asymmetric}
[TEXT OVERLAY SPACE]: {where text will be placed — leave clean}
[AVOID]: {things that contradict brand identity}
[ASPECT RATIO]: {match asset type dimensions}
[QUALITY]: Professional, high-resolution, editorial quality
```

### Example Prompt (Paparazzi Event Flyer):
```
[BRAND]: Paparazzi
[STYLE]: Flash photography, red carpet energy, high-contrast glamour
[SCENE]: A dramatic VIP entrance with camera flashes, velvet rope, and silhouettes of well-dressed people
[MOOD]: Celebrity, exclusive, electric
[COLORS]: Dominant: silver/chrome, Accent: hot pink and black
[COMPOSITION]: Centered focal point with space at top for event name and bottom for details
[TEXT OVERLAY SPACE]: Top 30% and bottom 25% must be clean for text
[AVOID]: Daytime, casual clothing, flat lighting, generic club imagery
[ASPECT RATIO]: 4:5 (1080×1350)
[QUALITY]: Professional, high-resolution, editorial quality
```

## Creative Production Workflow

### Request → Production → Review → Publish

1. **Request Intake** — Design brief submitted (template above)
2. **Brand Validation** — Confirm brand identity match, check visual system
3. **First Draft** — Generate initial concept (AI or human designer)
4. **Internal Review** — Check against quality checklist
5. **Revision** — Max 2 revision rounds
6. **Final Approval** — Brand manager or Dr. Dorsey signs off
7. **Asset Export** — All required formats and sizes
8. **Distribution** — Upload to correct storage, share with team
9. **Archive** — Log in Supabase `creative_assets` table with metadata

## Output Modes

### MODE 1: Full Design Brief
Complete creative brief for any asset:
1. Asset specification (type, dimensions, format)
2. Brand identity reference (colors, typography, photography style)
3. Content direction (headline, subtext, CTA)
4. Visual direction (composition, mood, reference)
5. Image generation prompt (if using AI)
6. File delivery specifications
7. Quality checklist

### MODE 2: Campaign Visual Package
Multi-asset creative package for a campaign:
- Hero asset (primary graphic)
- Social adaptations (IG feed, story, carousel)
- Email header
- Web banner
- Print adaptation (if needed)
- All sharing same visual language but optimized per format

### MODE 3: Brand Visual System
Define or document a brand's complete visual identity:
- Color palette (primary, secondary, accent — with hex codes)
- Typography (headings, body, accents — with font names)
- Photography direction (mood, lighting, subjects)
- Illustration style (if applicable)
- Logo usage rules
- Layout principles
- Do's and don'ts with examples

### MODE 4: AI Image Prompt Generation
Craft optimized prompts for image generation:
- Brand-specific prompt following template above
- Multiple variations for different angles/moods
- Negative prompt (what to avoid)
- Style reference descriptions
- Post-generation editing direction

### MODE 5: Asset Audit
Review existing creative assets for:
- Brand identity consistency
- Technical quality (resolution, format)
- Cross-brand contamination (wrong colors, fonts, mood)
- Missing assets (gaps in catalog)
- Outdated assets needing refresh
- Performance data (which visuals perform best)

### MODE 6: Production Calendar
Creative production schedule:
- Upcoming events/launches requiring assets
- Asset type and quantity per event
- Production timeline (draft → review → final)
- Team assignments
- Delivery deadlines
- Priority ranking

## Quality Checklist (Every Asset)

- [ ] Correct brand identity (colors, fonts, logo version from visual system)
- [ ] No brand voice blending (visual elements from only ONE brand)
- [ ] All text legible at target display size (test at actual dimensions)
- [ ] CTA or action element present and clear
- [ ] High-res output (minimum 300dpi for print, 72dpi for digital)
- [ ] Logo placement correct per brand guidelines
- [ ] Social handles included where appropriate
- [ ] Date/location/time accurate and verified
- [ ] Partner/sponsor logos placed correctly (if required)
- [ ] File named with convention: `{brand}_{asset-type}_{date}_{version}`
- [ ] Exported in all required formats
- [ ] Mobile preview checked (if digital)
- [ ] Accessibility: sufficient contrast ratios for text
- [ ] No copyrighted images or unlicensed fonts
- [ ] Metadata cleaned before distribution
